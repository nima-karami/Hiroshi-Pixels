Hiroshi Pixels is an interactive computer-generated art inspired by the works of Hiroshi Kawano (1925-2012), a Japanese pioneer of computer art.

This artwork generates a random matrix using variables including row count, column count, and variance. Then, the values in the matrix are visually represented with a number of distinct graphic styles developed for this artwork. Finally, a genetic algorithm is employed to change the value of each cell based on its most common neighbor.

Use the control panel on the bottom left of this page to customize this artwork to your liking!
